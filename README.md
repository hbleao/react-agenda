## Tecnologias
- React
- CRA
- Typescript
- Context Api
- Sass
- Axios
- Json server

## Comandos

Instalação:

```bash
npm install
# or
yarn
```

Ambiente de desenvolvimento:

```bash
npm run dev
# or
yarn dev
```

Ambiente de produção:

```bash
npm run start
# or
yarn start
```

Comando para geração build do projeto:

```bash
npm run build
# or
yarn build
```
